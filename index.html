About Us

QHSS Cyber is a web development club in Queens High School for the Sciences at York College (QHSS@YC). We create websites and learn the trending web dev technologies together!

You don't have to know anything about programming to join—just be willing to learn and dedicate your time to the club.

Want to Join?

Make sure to join our Discord and tag one of the board members!

Meetings are held every Wednesday in room 213 after 9th with Mr. Mondestin. They are mainly held for status updates, group planning, brainstorming, and workshops. Everything is announced in our Discord. Most of the work is done outside of club time or over Discord.

Why Web Development?

Web development combines coding and graphic design.

Through this club, you will:
  • Build a portfolio.
  • Learn cutting edge technology.
  • Look good to colleges and employers.

Our Current Tech Stack

Web Development

  • CSS (SASS and Bootstrap)
  • Javascript (JQuery and React)

Graphic Design

  • Adobe XD

Projects and Activities

For beginners

The main agenda we have for beginners is to learn the skills necessary for web development. We recommend:

  • Codeacademy for learning HTML and CSS
  • Scrimba for learning Bootstrap (CSS)
  • Net Ninja's tutorials for learning Sass (SCSS)
  • "You Don't Know JS" (E-Book) for learning Javascript
  • Codeacademy for learning NodeJS, Angular, and React.

For the experienced

This is where you get the real value from this club, through projects! Our current ongoing project is upgrading this website (currently maintained using GitHub at here) to a more unique design. However, we're open to new ideas, such as AI web apps, freelancing, etc.
